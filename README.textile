h1. Overview

This project will be based on:
* Spring 3.0.0 - REST annotations and component wiring
* Neo4j - graph database
* Jackson - JSON serialization

The intent is to create a connection based application that is accessible via REST calls. It would be used by a UI, with some experiments being amplesdk for the UI. 

h2. Goals


The goals of the inital version is to:
* create users
* register users
* add users to groups
* add friends to users
* send invites to users - allowing registration
* integration with third-party social apis

h2. Building and Running

Build uses maven 2, build by:
* mvn clean package

You can run with the following:
* mvn clean jetty:run
* go to social graph in browser "http://localhost:8080/socialgraph/":http://localhost:8080/socialgraph/
